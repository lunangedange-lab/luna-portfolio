# L<i class="fas fa-moon"></i>una Portfolio

Portfolio personnel de développeur avec thème lunaire.

## Structure

- **index.php** : Page d'accueil avec projets phares
- **about.php** : À propos avec compétences et parcours
- **projects.php** : Galerie de projets avec filtres
- **nature.php** : Galerie photo nature avec citations
- **contact.php** : Formulaire de contact et WhatsApp

## Technologies

- PHP 8+
- MySQL/MariaDB
- HTML5/CSS3
- JavaScript
- Font Awesome Icons

## Installation

1. Importer la base de données
2. Configurer `config/database.php`
3. Uploader sur votre hébergeur
4. Accéder via votre domaine

## Fonctionnalités

- Design responsive mobile-first
- Thème lunaire personnalisé
- Filtres interactifs
- Formulaire de contact avec WhatsApp
- Galeries photo animées
- Navigation fluide
