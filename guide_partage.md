# Guide de Partage de Portfolio L<i class="fas fa-moon"></i>una

## Partage Immédiat

### 1. Lien Local (pour test)
```
http://localhost/portfolio/
```

### 2. Partage WhatsApp
- Envoyer le lien local à tes contacts
- Ajouter une capture d'écran
- Expliquer que c'est une version de démo

### 3. Partage Facebook/Instagram
- Faire des captures d'écran magnifiques
- Poster avec le lien local
- Utiliser les hashtags : #portfolio #developpeur #webdesign

### 4. Partage LinkedIn
- Poster dans ta timeline
- Mentionner les technologies utilisées
- Ajouter le lien vers ta démo
